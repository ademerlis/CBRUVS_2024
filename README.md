This repository contains data and analysis scripts for the manuscript:

## [The secret life of coral predators: Documenting fish corallivory on restored corals in Florida using Coral-Baited Remote Underwater Video Systems (C-BRUVS)]
#### **Authors:** Allyson DeMerlis, Erin Weisman, Martine D'Alessandro, Bautista Tobias, Joseph Unsworth, Diego Lirman
#### **Journal:** _Bulletin of Marine Science_ 

-----

### Description:
This repository contains all data and code used to quantify corallivorous fish predation on outplanted coral fragments using Coral-Baited Remote Underwater Video Stations (C-BRUVS), and to compare C-BRUVS-derived predation observations to paired diver-based Reef Visual Census (RVC) fish surveys and coral condition surveys, at a reef site in Miami-Dade County, Florida.

### Contents:

#### Data (`data_clean/`):
* **CBRUVS_bites.csv:** C-BRUVS bite log of video analysis.
* **Diver_fish_survey.csv:** Diver-based fish survey data.
* **Diver_coral_survey.csv:** Diver-based coral fragment condition data.

#### Analysis:
* **CBRUVS_Manuscript.Rmd:** Reads the three clean CSVs from `data_clean/` and contains all code used to generate the manuscript figures (site map, Figures 2–5) and statistical results (parrotfish age-phase composition, coral species selectivity index, bite-frequency vs. tissue-mortality regression, GLMM of cumulative predation prevalence over time). Summary tables produced along the way are written to `output_tables/`.

#### Figures (`Figures/`):
* This folder contains all manuscript figures, generated by `CBRUVS_Manuscript.Rmd`.

#### Output Tables (`output_tables/`):
* Summary tables generated by `CBRUVS_Manuscript.Rmd` (predation summary by coral species, fish-coral selectivity index values, fragment bite summary, C-BRUVS/RVC fish size comparison, and GLMM post-hoc pairwise comparisons), provided here as a convenience alongside the manuscript's tables.
